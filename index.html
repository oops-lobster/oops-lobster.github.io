Sonmalgwan (손말관)

Web-based Korean Sign Language accessibility prototype

Built and deployed a web application that converts Korean text or speech input into sequences of Korean Sign Language dictionary videos. Integrated data/API from the Korean National Sign Language Dictionary and designed a context-analysis pipeline for segmenting Korean sentences, matching KSL expressions, and handling unmatched terms. Reached out to 20+ Deaf organizations, KSL researchers, educators, and interpreters to request linguistic review, usability testing, and bug feedback. Currently exploring bidirectional speech-sign communication and gamified KSL learning. Live project: handsigns.vercel.app.

Handsigns First Step (손말첫걸음)

Introductory KSL fingerspelling learning MVP

Built an introductory web service for first-time Korean Sign Language fingerspelling learners to study step by step using reference videos from the Korean National Sign Language Dictionary. Users move through a learning flow of watching, understanding, imitating, checking, and completing, with basic practice feedback supported by browser camera-based hand recognition. Designed as an educational MVP for families of Deaf and hard-of-hearing children and beginner learners to access KSL more easily, rather than as a complete sign-language classifier. Live project: handsignsfirststep.vercel.app.

KSL fingerspelling Dictionary reference videos Camera-based hand recognition Educational MVP Live
